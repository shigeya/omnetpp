Prerequisites

The OMNeT⁠+⁠+ IDE (and the OMNeT⁠+⁠+ simulation framework itself) requires a preinstalled compiler toolchain to function properly.

  • On Windows: The OMNeT⁠+⁠+ distribution comes with a preconfigured MinGW compiler toolchain (MinGW is the Windows port of the GCC compiler.) You do not have to install or download anything. The IDE should be able to use the MinGW compiler out of box.
  • On Linux: The default GCC toolchain can be used. You have to install the GCC toolchain on your system before trying to compile a simulation with OMNeT⁠+⁠+. Please read the Install Guide for detailed instructions.
  • On Mac OS X: You should install Xcode Developer Tools to get compiler support before trying to compile a simulation with OMNeT⁠+⁠+. Please read the Install Guide for detailed instructions.
