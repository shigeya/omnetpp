Workspaces

A workspace is basically a directory where all your projects are located. You may create and use several workspaces and switch between them as needed. During the first run, the OMNeT⁠+⁠+ IDE offers to open the samples directory as the workspace, so you will be able to experiment with the available examples immediately. Once you start working on your own projects, we recommend that you create your own workspace by selecting File | Switch Workspace | Other. You can switch between workspaces, as necessary. Please be aware that the OMNeT⁠+⁠+ IDE restarts with each switch in workspaces. This is normal. You can browse workspace content in the Project Explorer, Navigator, C/C++ Projects and similar views. We recommend using Project Explorer.
